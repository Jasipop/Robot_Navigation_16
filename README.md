## ğŸ“– Project Overview
This repository contains the source code, simulation assets, and navigation scripts for the **EBU6350 Mini-Project**. The goal is to develop an autonomous mobile robot capable of navigating a known maze topology using **LIDAR** and **Camera** sensor fusion.

The system is designed to work in both a **Gazebo simulation environment** and on the physical **Yahboom MicroROS-ESP32 robot**.
